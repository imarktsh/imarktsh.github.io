Linux命令

Linux相关

1.当前目录 ./ 切换目录 cd
2.查看当前进程 ps 执行退出 exit 查看当前路径pwd
3.清屏 clear 退出当前命名 ctrl+c 执行睡眠ctrl+z
4.查看当前作业jobs 列出进程的PID和作业号jobs -l
5.软链接 ln -s slink source 相当于快捷方式 硬链接ln link source 相当于copy一份不可跨区
6.创建目录mkdir 创建文件touch / vi 文件权限修改chmod
7.查看文件内容的命令：vi 编辑方式查看 可修改 cat 显示文件全部内容 more 、less分页显示 tail仅查看尾部 可指定行数 head 仅查看头部可指定行
8.删除文件 rm 删除目录下文件 rm -r 删除空文件夹 rmdir
9.对一个文件的内容进行统计 wc -c 统计字数 -l 统计行数 -w统计字数
10. grep强大的文本搜索工具，能使用正则表达式搜索文件
11. Linux进程的七种状态

12.查找系统支持所有信号 kill -l
13.查看磁盘使用情况df -hl
14.查看网络是否连通netstat
15.which只能查可执行文件 whereis只能查二进制文件 说明文档源文件等
16.Linux的静态库

17.nm工具作用 是linux下自带的特定文件分析工具，一般用来检查分析二进制文件、库文件、可执行文件中的符号表，返回二进制文件中各段的信息
第一列 当前符号的地址 第二列 当前符号的类型 第三列 当前符号名称

18.ldd工具作用 dd是用来分析程序运行时需要依赖的动态链接库的工具
第一列 程序需要依赖什么库 第二列 系统提供的与程序需要的库对应的库名称 第三列：依赖库加载的开始地址

19./proc目录 Linux内核提供了一种通过/proc文件系统，在运行时访问内核内部数据结构、改变内核设置的机制 存有进程运行时所有信息
20. 父进程先结束 子进程会变成孤儿进程 子进程先结束 父进程没有直到 父进程会变成僵尸进程。
